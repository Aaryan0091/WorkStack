# WorkStack

The intelligent bookmark and activity management platform.

---

## Overview

WorkStack helps you organize bookmarks, track browsing activity, and find content with AI-powered search. Just sign up and install the browser extension.

---

## How It Works

1. **Sign up** at workstack.app
2. **Download the extension** (Chrome, Edge, Brave)
3. **Start browsing** - bookmarks and activity are automatically saved

---

## Features

### 💾 Bookmark Management
- Save any URL with one click
- AI auto-tagging powered by Groq
- Organize with collections
- Full-text search

### 🧠 Semantic Search
- Search by meaning, not keywords
- "ferrari" finds "car", "sports car", "italian automotive"

### 📊 Activity Tracking
- Track time spent on websites
- Session history and restore
- Daily, weekly, monthly stats

### 👥 Sharing
- Share collections via public links
- Guest access for shared content
